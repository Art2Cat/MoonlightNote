# MoonlightNote
StudyJams进阶项目

MoonlightNote基于Firebase并遵循Material Design开发的一款简洁高效的记事本应用。


Features Include:
- Material Design UI 以及易用性
- 所有记事本内容存储于 Firebase Database(不存于本地)*
- 对记事本内容的分享
